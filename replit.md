# Mahmoud Sawaftah - Personal Portfolio Website

## Overview
A modern, responsive personal portfolio website built with vanilla HTML5, CSS3, and JavaScript. The website showcases Mahmoud Sawaftah's professional experience as a Call Center Operation Team Leader and Customer Service Specialist.

## Project Structure
```
├── index.html          # Main HTML structure with all sections
├── styles.css          # Complete CSS styling with responsive design
├── script.js           # JavaScript for interactivity and form validation
├── server.py           # Simple Python HTTP server
├── attached_assets/    # Images and assets
│   └── WhatsApp Image... # Profile photo
└── replit.md           # This documentation file
```

## Features
- **Hero Section**: Name, title, tagline, and call-to-action buttons
- **About Me**: Professional photo and description
- **Skills**: Visual progress bars for key skills
- **Experience**: Timeline-based professional history
- **Portfolio**: Project cards showcasing achievements
- **Contact Form**: JavaScript validation for all fields
- **Footer**: Social media links and contact info

## Technical Details
- **Layout**: CSS Flexbox and Grid
- **Animations**: Fade-in on scroll, smooth scrolling
- **Responsive**: Mobile-first design, works on all devices
- **Typography**: Inter font from Google Fonts
- **Icons**: Font Awesome 6.4

## Running the Website
The website is served using Python's built-in HTTP server on port 5000.
